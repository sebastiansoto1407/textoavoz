# Texto a Voz

Aplicación web estática para deletrear texto largo usando la API nativa SpeechSynthesis.

## Uso

Abre `index.html` en el navegador y pega cualquier texto. Puedes ajustar la velocidad y usar los botones de reproducir, pausar y detener.

## Características

- Diseño responsive para móvil, tablet y escritorio.
- Modo oscuro con interfaz moderna.
- Deletreo carácter por carácter.
- Espacios con pausa breve.
- Saltos de línea hablados como “Abajo”.
- Resaltado visual del carácter actual.
